Algumas tags de texto

Simbolo de tags

Simbolo de tags: <digite a tag aqui>
Basicamente, todo o HTML é baseado em tags
Você pode digitar comentários em <!-- comentário -->
Digite lorem para gerar um texto aleatório para testes.

Primeiras:

Título da aba: title (fica na parte de head do html)
Título de nível 1: h1 (h de heading - título, cabeçalho)
Parágrafo: p
Itálico: em ou i
Negrito: strong ou b
Marca-texto: mark
Sublinhado: ins ou u
Tachado (excluído): del

Quebra de texto

Quebra de linha (tipo \n): br (de break row)
Linha horizontal: hr

Essas tags não tem fechamento

Códigos fonte

Neste formato, a fonte monoespaçada - cada letra ocupa o mesmo espaço. Útil para digitar algorítmos

    
        num1 = int(input("Digite o valor de x: "))
        num2 = int(input("Digite o valor de y: "))
        if(num1 > num2):
            print(f'x é maior que y')
        elif (num1 < num2):
            print(f'x é menor que y')
        else:
            print('Os números são iguais')

  • 🥶 Código U+1F976
