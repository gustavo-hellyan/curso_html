Estilos básicos

Estilos inline: você escreve o estilo na própria tag. Esse método não é bom ou prático, mas vou usar para fazer as anotações e fazer estilos diferentes no mesmo documento html. Exemplo de estilo inline:

style="text-align: center;"

CSS interno: colocamos no estilo na parte de head e dentro da tag style: 

    <style> 

        h1{
            ...
        }
        p{
            ...
        }
    </style>
obs: h1 e p são exemplos de seletores.

CSS externo: O MAIS INDICADO. Você cria um arquivo externo .css e depois importa nos seus arquivos .html. É bem útil, pois você pode importar o mesmo estilo para diferentes documentos e manter um tema uniforme para o seu site. Observe na parte head o seguinte comando:

<link rel="stylesheet" href="style.css">

Você pode inclusive importar mais de um arquivo .css

Outros atributos

Texto centralizado: text-align: center;
texto justificado: text-align: justify;
"em" é o tamanho da fonte selecionado. Por exemplo, se temos font-size: 10px, font-size: 2em = 20px, 1.5em = 15px... px é uma unidade do HTML
