Tipografia - fontes

Anatomia dos tipos

a letra x minúscula é a primeira letra criada durante a criação de uma fonte. Ela define a altura base (chamada altura x) de todas as letras minúsculas.

Fontes SERIFADAS:



Cada letra é um GLIFO ou caractere e o conjunto de todos os glifos de uma família forma uma FONTE

As fontes podem ter diferentes maneiras de representar um glifo. Essas diferentes maneiras são as FAMÍLIAS TIPOGRÁFICAS

Categoria de fontes:

Dica: para documentos impressos use fontes serifadas, porém para textos grandes em tela, use fontes sans-serif.
Fontes displays: não se preocupam em seguir as regras de anatomia.

Não exagere botando muitas fontes no seu site. Geralmente temos três ou quatro (uma para título, uma de destaque e uma de texto normal)

Fonte no CSS

A fonte Arial é sem serifa.

Quando você escreve style="font-family: Arial, Helvetica, sans-serif;", você está definindo três fontes.
O navegador vai procurar primeiro Arial na máquina. Se não achar, vai procurar Helvetica. Se não achar, vai procurar qualquer fonte sans-serif do dispositivo (além do sans-serif temos também monospace, serif).
Isso é o que chamamos de Safe Font Combination. Você pode ver mais exemplos em CSS Web Safe Font Combinations

Caso o nome da sua fonte tenha duas palavras, escreva ele entre aspas

Normalmente, a fonte padrão do navegador é Times New Roman e o tamanho padrão: 16px.

Tamanho, peso e estilo (font-size, font-weight e font-style)

Medidas absolutas: cm, mm, in (polegadas), px (pixels), pt (ponto), pc (paica)
Medidas relativas: em (relativa ao tamanho atual da fonte), ex (relativo ao tamanho x da fonte), rem, vw, vh, %

Mais recomendável: use px e em. Não use as outras.

Se a fonte é 12px, 1 em = 12px, 2em = 24px, 0.5em = 6px...

Temos os pesos:

  • lighter
  • normal
  • bold
  • bolder

Estilos:

  • normal
  • itálico

Obs: note que aqui não estamos usando as tags em e strong

Shorthand font

Macete para configurar a fonte em uma só linha:

  • font: style weight size family
  • ex: font: italic lighter 12px Arial, sans-serif
  • ex: font: bold 14px Times New Roman, serif
