IDs e Classes

Usamos Id quando queremos personalizar a formatação de um único elemento (um paragráfo, título...) e usamos classe quando queremos personalizar (com a mesma formatação) mais de um elemento

id no HTML é # em CSS
class no html é . em CSS
: para pseudoclasse
:: para pseudoelemento
> para children (tags dentro de outras tags)

Dica: ao nomear a classe, não nomeie de acordo com a forma (exemplo, class='azul'), pois a forma pode ser alterada depois. Nomeie de acordo com a função (exemplo: class='destaque').

Tag DIV e SPAN: não faz nada específico. São tags usadas para aplicar estilos/formatações através classe ou id. A div é a nível de bloco e a span a nível de linha.

Podemos aplicar em um elemento um id e uma classe ao mesmo tempo e também mais de uma classe (class='class1 class2...')
