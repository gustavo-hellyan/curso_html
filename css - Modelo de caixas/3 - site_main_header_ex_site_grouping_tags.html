Explicação

Veja o arquivo html. Não estamos usando divs, mas as tags header (cabeçalho, não confundir com head), main, footer (rodapé), section, nav, article, aside. Elas tem a mesma função do div, porém deixam o nosso código html mais semântico, em um formato melhor para colocar o conteúdo do seu site. Ainda podemos editar melhor o estilo do seu site! Veja o CSS.

Veja que podemos alterar a formatação de uma tag dentro de outra. Exemplo: header > h1 altera os h1 dentro de header enquanto main > h1 altera os h1 dentros da main. Nesse caso, nem foi preciso aplicar ids.
